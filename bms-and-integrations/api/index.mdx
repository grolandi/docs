## Introduction

Greetings and welcome to the Public API by inBiot! We are delighted to provide you with the opportunity to maximize the potential of your device. With our API, you can effortlessly access your MICA data and create an unlimited number of applications.

## Authentication

To get started, all you need is an `apiKey` that uniquely identifies your MICA. You can generate and manage these API Keys for all your devices through My inBiot platform.

## API Usage

There are several things to take into account:

- API Key is personal and non-transferable. You must be sure of storing it properly to ensure security.
- Response format is JSON.
- Access to API is limited to 6 requests per hour, and this restriction applies to each end-point and system individually.
- Each system has a restricted access of 6 requests per hour in all of its endpoints. This limitation applies to each system.
